# Curso de Preparación para la Certificación LPIC (Linux Professional Institute)

![General](<Módulo 0 - Presentación e Infraestructura/imgs/general.png>)

Bienvenido al repositorio del curso de preparación para la certificación LPIC. 
Este repositorio contiene todos los materiales, manuales y recursos necesarios divididos por módulos para adquirir los conocimientos de administración de sistemas Linux.

## Estructura del Curso

- **Módulo 0**: Presentación e Infraestructura
- **Módulo 1**: Introducción a Linux
- **Módulo 2**: Instalación, Configuración y Gestión Ubuntu Server
- **Módulo 3**: Comandos de Administración
- **Módulo 4**: Gestión de Dispositivos y Sistemas de Fichero
- **Módulo 5**: Entornos Shell y Scripting
- **Módulo 6**: Interfaces de Usuario
- **Módulo 7**: Tareas Administrativas
- **Módulo 8**: Servicios Básicos
- **Módulo 9**: Configuración y Gestión de Red
